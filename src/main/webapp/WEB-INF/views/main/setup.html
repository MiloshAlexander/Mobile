The administrator has already been initialized.

Before you start using the application, you must initialize the administrator. To do this, you must enter username and password.

Also, roles that are used in this application are automatically initialized: ROLE_ADMIN, ROLE_USER, ROLE_ABONENT.

20 phone numbers will be automatically added to the database.

After that you can go to the personal administrator page where you can perform the following operations:
  - view all existing tariffs and add a new one;
  - view all existing services and add a new one;
  - view all subscribers, as well as their personal information.
